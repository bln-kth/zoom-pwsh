# zoom-dashboard-mini
Creates zoom-dashboard-mini.html

### api url
* US = `api.zoom.us`
* EU = `eu01web.zoom.us`

## To run
* Install/Download python:
https://www.python.org/downloads/
* Install/Download pip
https://pip.pypa.io/en/stable/installing/
* Install deps:
`pip install pyjwt`
* Run the script:
`python dashboard-mini.py`
