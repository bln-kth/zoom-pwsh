# usage-report
Show usage report of current year and month.

### api url
* US = `api.zoom.us`
* EU = `eu01web.zoom.us`

## To run
* Install/Download PowerShell: https://docs.microsoft.com/en-us/powershell/scripting/install/installing-powershell
* Install deps:
`Install-Module PSZoom`
* Run the script:
`usage-report.ps1`
